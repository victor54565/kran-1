# 🏗️ Клуб Автокрановщиков Костаная

Веб-платформа для объединения профессиональных крановщиков. Здесь можно найти коллег, обмениваться опытом, предупреждать о недобросовестных работодателях и вести прозрачный учет финансов.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Или с автоматической перезагрузкой (если установлен nodemon):
```bash
npm run dev
```

### 3. Открыть сайт
Перейди на `http://localhost:3000` в браузере.

## 📋 Что делает каждая страница

### 🔧 **Коллеги** (`/colleagues.html`)
- База данных членов клуба с техникой
- Каждый может добавить себя с указанием своих кранов (тип, тоннаж, вылет)
- Прямые контакты без посредников

### 🛒 **Барахолка** (`/marketplace.html`)
- Объявления о продаже/покупке/обмене запчастей, тросов, масла и т.д.
- Пользователи сами добавляют и удаляют свои объявления
- Админы могут удалять нарушающие объявления
- Фильтрация по категориям

### 🚫 **Черный список** (`/blacklist.html`)
- База недобросовестных работодателей
- Информация о нарушениях (задержка оплаты, нарушение ТБ и т.д.)
- Только просмотр для обычных пользователей
- Админы могут добавлять и удалять записи

### 📚 **Материалы** (`/materials.html`)
- Полезные видеоруководства с YouTube
- Статьи о технике безопасности, ремонте
- PDF документы с инструкциями
- Управление только для админов

### 💰 **Касса** (`/treasury.html`)
- Прозрачный учет финансов клуба
- Приходы (взносы членов)
- Расходы (оплата сервера, домена и т.д.)
- Полная история всех операций
- Автоматическое обновление каждые 10 секунд

### 🔐 **Админ панель** (`/admin.html`)
- Управление материалами (добавление/удаление видео и статей)
- Управление кассой (добавление приходов и расходов)
- Управление черным списком
- Управление участниками клуба
- **Пароль: `12345`** (поменяй в `data/admins.json`)

## 🔌 API Endpoints

### Коллеги
- `GET /api/colleagues` - получить всех
- `POST /api/colleagues` - добавить нового
- `DELETE /api/colleagues/:id` - удалить (только админ)

### Барахолка
- `GET /api/marketplace` - получить все объявления
- `POST /api/marketplace` - создать объявление
- `DELETE /api/marketplace/:id` - удалить объявление

### Черный список
- `GET /api/blacklist` - получить все записи
- `POST /api/blacklist` - добавить запись
- `DELETE /api/blacklist/:id` - удалить (только админ)

### Касса
- `GET /api/treasury` - получить всю историю
- `POST /api/treasury` - добавить запись (только админ)
- `DELETE /api/treasury/:id` - удалить запись (только админ)

### Материалы
- `GET /api/materials` - получить все
- `POST /api/materials` - добавить (только админ)
- `DELETE /api/materials/:id` - удалить (только админ)

### Аутентификация
- `POST /api/admin/login` - получить JWT токен

## 🔐 Безопасность

1. **Админ пароль** - хранится в `data/admins.json`
2. **JWT токены** - используются для защиты админ-запросов
3. **Токен хранится** - в localStorage браузера

**⚠️ ВНИМАНИЕ:** Для продакшена нужно:
- Поменять SECRET_KEY в server.js
- Использовать HTTPS
- Переместить пароли в переменные окружения (.env)
- Добавить валидацию входных данных

## 📝 Как менять пароль админа

Отредактируй `data/admins.json`:
```json
[
  {
    "id": 1,
    "name": "Admin",
    "password": "новый-пароль"
  }
]
```

## 🐛 Проблемы и решения

**Ошибка: "Port 3000 already in use"**
- Убей процесс на этом порту или поменяй порт в server.js

**Данные не сохраняются**
- Проверь, что папка `data/` существует и приложение имеет права на запись

**Админ панель не открывается**
- Проверь консоль браузера (F12) на ошибки
- Убедись, что сервер запущен
